Count A

Ecrire un programme dans un fichier C count-A.c reçoit un argument de ligne de commande et qui affiche le nombre d'occurrences de la lettre 'A' dans cette chaine. En cas de problème, votre programme doit s'arrêter avec le code d'erreur 42.

Exemples

Exemple 1

entrée : $ ./count-A AAAAAAAA sortie : 8

Exemple 2

entrée : $ ./count-A Alphabet sortie : 1
