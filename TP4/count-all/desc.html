Count A

Ecrire un programme dans un fichier C count-all reçoit un argument de ligne de commande et qui affiche le nombre d'occurrences de chacune des 26 lettres de l'alphabet. Votre programme ne fera pas la différence entre les lettes majuscules et minuscules. Les autres caractères (ponctuation, accents, etc.) seront ignorés.

En cas de problème, votre programme doit s'arrêter avec le code d'erreur 42.

Exemples

Exemple 1

entrée : $ ./count-all abcdef sortie : a:1 b:1 c:1 d:1 e:1 f:1 g:0 h:0 i:0 j:0 k:0 l:0 m:0 n:0 o:0 p:0 q:0 r:0 s:0 t:0 u:0 v:0 w:0 x:0 y:0 z:0
