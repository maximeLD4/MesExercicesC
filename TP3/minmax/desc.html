Min Max

Ecrire dans le fichier d'entête "minmax.h" une fonction minmax qui permet de récupérer le minimum et le maximum d'un tableau d'entier passé en argument:

Le prototype de la fonction doit être: c void minmax(int *arr, int length, int *min, int *max)

Vous pouvez tester votre fonction modifiant le programme se trouvant dans le fichier main.c.
